# Mixed-Use Development Case Studies

A portfolio of mixed-use development case studies across Texas, built as an interactive static website.

## Projects

| City | Project |
|------|---------|
| Dallas, TX | Mixed-Use Development |
| Austin, TX | Waterfront Development |
| Houston, TX | Transit-Oriented District |
| San Antonio, TX | Arts District |
| Fort Worth, TX | Rail Yard District |
| Frisco, TX | Innovation Hub |
| Plano, TX | Legacy TOD |
| Plano, TX | Assembly Park |
| El Paso, TX | Borderlands District |

## Stack

- Vanilla HTML/CSS + React 18 (via CDN, no build step)
- Space Grotesk + Space Mono (Google Fonts)
- Isometric SVG visualizations
- Zero dependencies, no bundler required

## Running Locally

```bash
npx serve .
```

Then open `http://localhost:3000`.
